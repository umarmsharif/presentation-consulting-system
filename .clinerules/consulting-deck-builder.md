# consulting-deck-builder

When the user asks to build a deck, a presentation, or to turn analysis into slides, follow the consulting-deck-builder skill: read `SKILL.md`, then `references/design-system.md`, `references/patterns.md`, and `references/charts.md`, and use `examples/build_demo.js` as a working pptxgenjs template.

Always run the intake questions (audience, goal, deck type, framing, length, theme, font, chrome) and get ghost-deck sign-off — action titles + layout per slide — before building any slide.
